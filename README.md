# BookLogex MVP

AI-powered bookkeeping & payroll app for Australia and New Zealand.

## Tech Stack
- Next.js (frontend)
- TailwindCSS + shadcn/ui
- Node.js backend
- Supabase/Postgres (database)
- Stripe (billing)
- OpenAI API (AI categorisation)

## Setup
1. Clone the repo
2. Install dependencies: `pnpm install`
3. Run locally: `pnpm dev`
